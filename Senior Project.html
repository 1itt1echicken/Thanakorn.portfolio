Environment Monitoring and Controling actuator mobile robot

Senior Project

หุ่นยนต์เคลื่อนที่อัตโนมัติเพื่อตรวจวัดและควบคุมสภาพแวดล้อม

ภาพรวมโครงการ

โครงงาน Environment Monitoring and Controlling Actuator Mobile Robot เป็นโครงงานวิศวกรรมที่พัฒนาขึ้นเพื่อตอบสนองความต้องการในการตรวจสอบคุณภาพอากาศและสภาพแวดล้อมในพื้นที่ปิดอย่างอัตโนมัติ โดยมีวัตถุประสงค์เพื่อให้บุคคลากร และผู้มีที่มีส่วนรับผิดชอบในการดูแลสถานที่สามารถจัดการกับสภาพแวดล้อมที่ผิดปกติได้อย่างทันท่วงที เพื่อที่จะสามารถลดความรุนแรงของปัญหาที่อาจจะเกิดขึ้น ซึ่งคาดว่าจะช่วยให้ผู้ที่ใช้งาน พื้นที่บริเวณนั้นเกิดความปลอดภัยมากยิ่งขึ้น

หุ่นยนต์สามารถเคลื่อนที่อัตโนมัติไปในพื้นที่ต่างๆ ตามเส้นทางที่กำหนด เพื่อตรวจวัดค่าสภาพแวดล้อมในจุดต่างๆ จากนั้นจะแสดงผลเป็นค่าของสภาพแวดล้อม ณ บริเวณนั้นแบบเรียลไทม์ และแสดงภาพรวมของสภาพอากาศในบริเวณนั้นออกมาเป็นลักษณะของแผ่นที่ความร้อน(Heat map) ซึ่งแสดงขึ้นบน Dashboard นอกจากนี้ยังสามารถควบคุมพัดลมระบายอากาศ เพื่อระบายอากาศเมิ่ออากาศมีความเป็นพิษ และเครื่องเพิ่มความชื้น เพื่อเพิ่มความชื้นในอากาศเมื่ออากาศแห้ง

รายละเอียดทางเทคนิค

ฮาร์ดแวร์

ส่วนประกอบด้านฮาร์ดแวร์ของหุ่นยนต์ที่ใช้ในการตรวจวัดและควบคุมสภาพแวดล้อม

องค์ประกอบหลัก:

  • หุนยนต์ใช้หุ่น ironx ซึ่งได้รับการสนับสนุนจาก TESR Co.,LTD.
  • ESP32 Microcntroller
  • เซนเซอร์วัดอุณหภูมิและความชื้น DHT11
  • เซนเซอร์วัดฝุ่นละออง PM2.5 และ PM10
  • เซนเซอร์ตรวจจับแก๊ส CO, CO2, และ VOCs
Arduino Sensors

ซอฟต์แวร์

ระบบซอฟต์แวร์ที่ใช้ในการควบคุมการทำงานของหุ่นยนต์และการประมวลผลข้อมูล

คุณสมบัติหลัก:

  • ระบบนำทางอัตโนมัติด้วย SLAM (Simultaneous Localization and Mapping) ไปยังจุดต่างๆ
  • การเก็บและประมวลผลข้อมูลจากเซนเซอร์ต่างๆ
  • การสื่อสารโดยใช้ MQTT Protocol
  • ระบบตรวจจับค่าผิดปกติแบบอัตโนมัติ
Python ROS2 C++ MQTT Protocol

แดชบอร์ดแสดงผล

ระบบแดชบอร์ดสำหรับแสดงผลข้อมูลการตรวจวัดและควบคุมอุปกรณ์

คุณสมบัติหลัก:

  • แสดงผลข้อมูลแบบเรียลไทม์ผ่านเว็บแอปพลิเคชัน
  • แผนที่ความร้อน (Heat Map) แสดงค่าต่างๆ ในแต่ละพื้นที่
  • ส่วนควบคุมอุปกรณ์ระบายอากาศและเพิ่มความชื้น
Node red

ผลลัพธ์และประโยชน์

โครงงานนี้ได้พัฒนาระบบหุ่นยนต์ที่สามารถตรวจวัดและควบคุมสภาพแวดล้อมได้อย่างมีประสิทธิภาพ โดยมีผลลัพธ์และประโยชน์ดังนี้:

  • การตรวจจับที่แม่นยำ - สามารถวัดค่าอุณหภูมิได้แม่นยำ ...°C, ความชื้น ...%RH และฝุ่น PM2.5 ได้แม่นยำ ...% เมื่อเทียบกับเครื่องวัดมาตรฐาน (กำลังทดสอบ)
  • ระบบทำงานอัตโนมัติ - สามารถเคลื่อนที่และตรวจวัดโดยอัตโนมัติ
  • การแสดงผล - สามารถแสดงค่าต่างๆที่วัดได้ และ heat map บนห้าจอ dashboard
  • การควบคุมอุปกรณ์ - ควบคุมพัดลมระบายอากาศและเครื่องเพิ่มความชื้นได้อัตโนมัติเมื่อจำเป็น
