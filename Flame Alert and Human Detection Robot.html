Internship Project

โครงการย่อยที่พัฒนาระหว่างการฝึกงาน โดยใช้ Visual Studio C#

ภาพรวมโครงการ

ในระหว่างการฝึกงานที่บริษัท [ชื่อบริษัท] ฉันได้รับมอบหมายให้พัฒนาโปรแกรมย่อย 3 โครงการเพื่อฝึกทักษะการใช้งาน Visual Studio และภาษา C# สำหรับการพัฒนาแอปพลิเคชันเดสก์ท็อป โดยมีจุดประสงค์เพื่อเรียนรู้หลักการพัฒนาซอฟต์แวร์ในสภาพแวดล้อมการทำงานจริง

โครงการทั้งหมดได้รับการพัฒนาโดยใช้ .NET Framework และได้รับคำแนะนำจากพี่เลี้ยงที่มีประสบการณ์ในด้านการพัฒนาซอฟต์แวร์ ซึ่งช่วยให้ฉันได้เรียนรู้แนวทางปฏิบัติที่ดีในการเขียนโค้ด การออกแบบส่วนติดต่อผู้ใช้ และการจัดการฐานข้อมูล

โปรแกรมคำนวณพื้นฐาน

แอปพลิเคชันเครื่องคิดเลขที่สามารถทำการคำนวณพื้นฐานและคำนวณทางวิทยาศาสตร์ได้หลากหลายฟังก์ชัน

คุณสมบัติหลัก:

  • การคำนวณพื้นฐาน: บวก, ลบ, คูณ, หาร
  • ฟังก์ชันทางวิทยาศาสตร์: sin, cos, tan, log, ln
  • บันทึกประวัติการคำนวณ
  • สามารถแปลงหน่วยพื้นฐานได้
C# WinForms .NET Framework

ระบบจัดการฐานข้อมูลสินค้า

แอปพลิเคชันที่ช่วยในการจัดการสินค้าคงคลัง บันทึกการเข้า-ออกของสินค้า และดูรายงานต่างๆ

คุณสมบัติหลัก:

  • เพิ่ม, แก้ไข, ลบข้อมูลสินค้า
  • บันทึกประวัติการเข้า-ออกของสินค้า
  • แจ้งเตือนเมื่อสินค้าใกล้หมด
  • สร้างรายงานสรุปสินค้าคงเหลือ
C# WPF SQL Server Entity Framework

แอปพลิเคชันจัดการตารางนัดหมาย

ระบบจัดการตารางนัดหมาย ที่ช่วยให้ผู้ใช้สามารถบันทึก, จัดการ และได้รับการแจ้งเตือนเกี่ยวกับนัดหมายต่างๆ

คุณสมบัติหลัก:

  • สร้างและจัดการนัดหมาย
  • แสดงปฏิทินรายวัน, รายสัปดาห์, รายเดือน
  • ตั้งค่าการแจ้งเตือนล่วงหน้า
  • ระบบค้นหาและกรองนัดหมาย
C# WPF SQLite MVVM

สิ่งที่ได้เรียนรู้

การพัฒนาโครงการย่อยทั้ง 3 นี้ ทำให้ฉันได้เรียนรู้ทักษะที่สำคัญหลายอย่างในการพัฒนาซอฟต์แวร์:

  • การออกแบบส่วนติดต่อผู้ใช้ที่ใช้งานง่ายและมีประสิทธิภาพ
  • การเขียนโค้ดตามหลักการ Object-Oriented Programming
  • การทำงานกับฐานข้อมูลทั้งแบบ SQL และ NoSQL
  • การจัดการกับข้อผิดพลาดและการทดสอบซอฟต์แวร์
  • การทำงานภายใต้การดูแลของพี่เลี้ยงในสภาพแวดล้อมการทำงานจริง
